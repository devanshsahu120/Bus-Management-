# 🚌 Bus Management System

A smart Bus Management System for colleges and schools to manage transportation efficiently. It includes features like live tracking, schedules, digital bus pass, and complaint system.

---

## ✨ Features

- Live Bus Tracking  
- Bus Schedule Management  
- Driver & Bus Details  
- Digital Bus Pass  
- Student ID Registration  
- Complaint System  
- Admin Login Panel  
- Real-Time Bus Status  

---

## 🚀 How to Run

1. Clone the repository  
git clone git@github.com:devanshsahu120/Bus-Management-.git  

2. Open folder  
cd Bus-Management-  

3. Install dependencies  
npm install  

4. Run server  
node server.js  

5. Open in browser  
http://localhost:3000
